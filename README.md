# C++ Programming Basics
This repository contains my daily practice for the CoSc1012 module at Addis Ababa University.

## Problems Included:
1. **Variable Swapping:** Swapping two numbers without using a temporary variable.
2. **Temperature Conversion:** Converting Celsius to Fahrenheit.
3. **Geometry:** Calculating the Area of a circle and Perimeter of a rectangle.
4. **Unit Conversion:** Converting Kilometers to Meters.

## Tools Used:
* Language: C++
* Concepts: #define macros, arithmetic operators, and basic I/O.
